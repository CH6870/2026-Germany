# 羅曼蒂克大道 (Romantic Road) 完整景點清單

以下依您從羅曼蒂克大道**由北至南**的順序，將全部27個城市及主要景點排列。景點前的方框供您勾選，星級評價 (★☆☆☆☆ 至 ★★★★★) 依網路好評與重要性排序，方便您規劃行程。

### 1. 符茲堡 Würzburg (起點)

- [X] ★★★★★ **符茲堡主教宮 (Würzburg Residence)** (巴洛克建築的傑作，擁有世界最大的濕壁畫天花板，聯合國世界遺產)
- [X] ★★★★☆ **舊美茵橋 (Alte Mainbrücke)** (橋上佇立著聖人雕像，可在此品嚐一杯葡萄酒，遠眺瑪麗恩堡要塞)
- [X] ★★★☆☆ 瑪麗恩堡要塞 (Festung Marienberg) (俯瞰符茲堡全景和葡萄園的絕佳地點)

### 2. 陶伯比紹夫斯海姆 Tauberbischofsheim

- [ ] ★☆☆☆☆ 途經小鎮 (主要為路過，可欣賞田園風光)

### 3. 勞達-柯尼希斯霍芬 Lauda-Königshofen

- [ ] ★☆☆☆☆ 途經小鎮 (葡萄酒產區的小鎮)

### 4. 巴特梅爾根特海姆 Bad Mergentheim

- [ ] ★★☆☆☆ 溫泉小鎮 (以溫泉療養和條頓騎士團城堡聞名)

### 5. 魏克斯海姆 Weikersheim

- [ ] ★★★☆☆ **魏克斯海姆宮 (Schloss Weikersheim)** (保存完好的文藝復興時期宮殿與巴洛克式花園)

### 6. 勒廷根 Röttingen

- [ ] ★☆☆☆☆ 途經小鎮 (歐洲第一個被官方認證的陽光小鎮)

### 7. 克雷格林根 Creglingen

- [ ] ★★☆☆☆ **主祭壇 (Herrgottskirche)** (收藏有著名木雕家 Tilman Riemenschneider 的聖母瑪利亞祭壇)

### 8. 羅騰堡 Rothenburg ob der Tauber

- [X] ★★★★★ **古城牆漫步 (Medieval City Wall)** (德國保存最完整的城牆，可環繞舊城區，俯瞰紅瓦屋頂)
- [X] ★★★★★ **普雷萊茵廣場 (Plönlein)** (羅騰堡最經典的明信片角落，Y字形街道的童話景觀)
- [X] ★★★★☆ 市集廣場與市政廳 (Marktplatz & Rathaus) (小鎮的中心，可登上塔樓欣賞全景)
- [X] ★★★☆☆ 羅騰堡聖誕博物館 (German Christmas Museum) (全年都能感受德國聖誕氛圍的夢幻之地)

### 9. 希靈斯菲斯特 Schillingsfürst

- [ ] ★☆☆☆☆ 途經小鎮 (擁有一座巴洛克風格的城堡)

### 10. 福伊希特旺根 Feuchtwangen

- [ ] ★★☆☆☆ 途經小鎮 (擁有一個美麗的市集廣場和修道院)

### 11. 丁克爾斯比爾 Dinkelsbühl

- [X] ★★★★☆ **舊城區漫步 (Old Town Walk)** (與羅騰堡齊名，但遊客較少，更能感受中世紀的寧靜氛圍)

### 12. 瓦勒施泰因 Wallerstein

- [ ] ★☆☆☆☆ 途經小鎮 (可遠眺瓦勒施泰因岩)

### 13. 諾德林根 Nördlingen

- [X] ★★★★☆ **登上「丹尼爾」塔樓 (Climb the "Daniel")** (從聖喬治教堂的塔樓俯瞰建在隕石坑中的完整圓形小鎮)
- [ ] ★★★☆☆ 環形城牆步道 (Circular Town Wall) (另一座擁有完整城牆的城市，可沿牆散步)

### 14. 哈爾堡 Harburg

- [ ] ★★★☆☆ **哈爾堡城堡 (Burg Harburg)** (德國最大、保存最完好的城堡之一)

### 15. 多瑙沃特 Donauwörth

- [ ] ★★☆☆☆ 帝國大街 (Reichsstraße) (色彩繽紛的建築構成美麗的街道)

### 16. 奧格斯堡 Augsburg

- [ ] ★★★★☆ **富格社區 (Fuggerei)** (世界上現存最古老的社會住宅區，年租金至今仍不到1歐元)
- [ ] ★★★☆☆ 奧格斯堡市政廳與黃金大廳 (Augsburger Rathaus & Goldener Saal) (德國文藝復興時期最重要的建築之一)

### 17. 弗里德貝格 Friedberg

- [ ] ★☆☆☆☆ 途經小鎮 (巴伐利亞的古老公爵城)

### 18. 萊希河畔蘭茨貝格 Landsberg am Lech

- [ ] ★★☆☆☆ 舊城區 (擁有美麗的河畔風光和色彩繽紛的房屋)

### 19. 霍恩夫爾希 Hohenfurch

- [ ] ★☆☆☆☆ 途經小鎮 (典型的巴伐利亞鄉村)

### 20. 雄高 Schongau

- [ ] ★★☆☆☆ 舊城區與城牆 (擁有部分保存完好的城牆和塔樓)

### 21. 派廷 Peiting

- [ ] ★☆☆☆☆ 途經小鎮 (位於阿爾卑斯山腳)

### 22. 羅滕布赫 Rottenbuch

- [ ] ★★☆☆☆ 修道院教堂 (前奧古斯丁修道院，內部裝飾華麗)

### 23. 維爾特施泰格 Wildsteig

- [ ] ★☆☆☆☆ 途經小鎮 (風景優美的鄉村地區)

### 24. 施泰因加登 Steingaden

- [X] ★★★★★ **維斯朝聖教堂 (Wieskirche)** (外觀樸實，內部極盡洛可可風格之華麗，聯合國世界遺產)

### 25. 哈爾布萊希 Halblech

- [ ] ★☆☆☆☆ 途經小鎮 (前往新天鵝堡途中的阿爾卑斯山麓村莊)

### 26. 施旺高 Schwangau

- [X] ★★★★★ **新天鵝堡 (Schloss Neuschwanstein)** (羅曼蒂克大道的終極夢幻景點，迪士尼城堡原型)
- [X] ★★★★★ **瑪麗安橋 (Marienbrücke)** (拍攝新天鵝堡全景的最佳地點，冬季需確認是否開放)
- [X] ★★★☆☆ 高天鵝堡 (Schloss Hohenschwangau) (路德維希二世的童年居所，更具生活氣息)

### 27. 菲森 Füssen (終點)

- [X] ★★★☆☆ **菲森老城區 (Füssen Altstadt)** (色彩繽紛的巴伐利亞小鎮，羅曼蒂克大道的完美句點)
- [ ] ★★☆☆☆ 萊希瀑布 (Lechfall) (位於城郊的壯觀階梯式瀑布)
