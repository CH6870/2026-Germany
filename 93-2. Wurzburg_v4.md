# 符茲堡基地 5天4夜 深度行程規劃 (V4 - 完整版)

本行程以**符茲堡(Würzburg)**作為固定住宿基地，採用「不換飯店」的輕鬆旅行方式。從哈修塔特出發，中途在Ingolstadt Village購物，抵達符茲堡後進行4天的法蘭肯地區深度探索，每日從符茲堡出發進行當日來回的一日遊，最後前往海德堡。此方案避免頻繁換飯店的疲累，讓您的自駕與火車行程更加流暢與舒適。

## 行程總覽表

| 日期                | 行程路線                                       | 當日活動重點與住宿規劃                                                                                                                                                               |
| :------------------ | :--------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1/28 (三)** | Hallstatt → Ingolstadt Village → 符茲堡 | **購物與移動日**：上午從哈修塔特出發，中途在Ingolstadt Village停留3小時購物，傍晚抵達符茲堡並辦理入住。<br>**住宿**: 符茲堡市中心飯店（連住4晚）                                           |
| **1/29 (四)** | 符茲堡 ↔ 班堡 (當日來回)                          | **世界遺產小鎮與煙燻啤酒**：搭乘火車前往「小威尼斯」班堡，逛週市集（7:00-18:00），深度參觀大教堂、玫瑰園、米歇爾山修道院，品嚐煙燻啤酒。<br>**住宿**: 符茲堡市中心飯店                                       |
| **1/30 (五)** | 符茲堡 ↔ 紐倫堡 (當日來回)                               | **帝國之城探索**：搭乘火車前往紐倫堡，遊覽皇帝堡、主市集廣場（7:00-20:00），品嚐紐倫堡香腸。<br>**住宿**: 符茲堡市中心飯店 |
| **1/31 (六)** | 符茲堡 ↔ 羅騰堡 (當日來回) | **羅曼蒂克大道精華**：自駕前往羅騰堡，逛週市集（8:00-12:00），漫步古城牆，品嚐雪球甜點，參觀聖誕博物館。<br>**住宿**: 符茲堡市中心飯店                                                                       |
