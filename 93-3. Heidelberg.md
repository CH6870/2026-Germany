# 海德堡 3天2夜 深度行程規劃 (Day 7-9)

本行程以**海德堡 (Heidelberg)** 作為住宿基地，涵蓋 2026 年 2 月 1 日至 2 月 3 日的完整規劃。包含海德堡市區深度遊、以及哈瑙/法蘭克福一日遊。此方案採用「不換飯店」的輕鬆旅行方式，讓您充分體驗德國浪漫大學城的魅力。

## 行程總覽表

| 日期 | 行程路線 | 當日活動重點與住宿規劃 |
| :--- | :--- | :--- |
| **2/1 (日)** | 符茲堡 → 海德堡（約 130 公里，1.5 小時） | **移動日**：15:00 從符茲堡出發，17:00 抵達海德堡並辦理入住。晚間前往老城區晚餐 + 老橋夜景。<br>**住宿**: Leonardo Hotel Heidelberg City Center（連住 3 晚） |
| **2/2 (一)** | 海德堡市區步行 | **大學城深度遊**：上午城堡導覽（巨型酒桶、藥房博物館），下午學生監獄 + Hauptstraße 購物。<br>**住宿**: Leonardo Hotel Heidelberg City Center |
| **2/3 (二)** | 海德堡 → 哈瑙 → 法蘭克福 → 海德堡（自駕） | **格林童話與金融之都**：上午哈瑙 Philippsruhe 宮童話展，下午法蘭克福羅馬廣場、歌德故居。<br>**住宿**: Leonardo Hotel Heidelberg City Center |

## 必訪景點研究與詳細規劃

### 📍 海德堡城堡 (Heidelberg Castle)

| 項目 | 資訊 |
| :--- | :--- |
| **地址** | Schlosshof 1, 69117 Heidelberg |
| **冬季開放時間** | 10:00-16:00（最後入場 15:30） |
| **門票** | €9（含纜車來回 + 城堡庭院 + 巨型酒桶 + 藥房博物館） |
| **導覽** | 英文導覽 €6（需預約） |
| **建議停留時間** | 2.5-3 小時 |

**必看亮點**：
- ✅ **巨型酒桶 (Großes Fass)**：世界最大的葡萄酒桶，可容納 221,726 公升
- ✅ **德國藥學博物館**：展示數百年的藥房歷史
- ✅ **城堡庭院**：俯瞰海德堡老城區和內卡河
- ✅ **伊莉莎白門**：浪漫的愛情故事遺跡

**交通方式**：
- 搭乘 Bergbahn 纜車（約 3 分鐘）
- 或從 Kornmarkt 步行上山（約 15-20 分鐘）

### 📍 老橋 (Alte Brücke)

| 項目 | 資訊 |
| :--- | :--- |
| **建造年份** | 1788 年 |
| **門票** | 免費 |

**拍照建議**：
