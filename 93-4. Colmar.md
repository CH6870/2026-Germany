# 科爾馬基地 3天2夜 行程規劃 (2/4-2/6)

本行程以**科爾馬 (Colmar)**作為固定住宿基地，深度探索法國亞爾薩斯地區的童話小鎮風情。從海德堡出發，途經史派爾大教堂，傍晚抵達科爾馬後進行2天的亞爾薩斯深度探索，包括科爾馬舊城區、小威尼斯、埃吉桑姆等景點，最後前往蒂蒂湖並銜接加米許行程。

## 行程總覽表

| 日期 | 行程路線 | 當日活動重點與住宿規劃 |
|:---|:---|:---|
| **2/4 (三)** | 海德堡 → 史派爾 → 科爾馬 | **移動日與德法跨境**：上午從海德堡出發，中途在史派爾大教堂短停（1.5小時），傍晚抵達科爾馬，夜遊小威尼斯燈光夜景。<br>**住宿**: Hôtel Le Colombier 或 Grand Hotel Bristol（連住2晚） |
| **2/5 (四)** | 科爾馬 ↔ 埃吉桑姆 (當日來回) | **亞爾薩斯童話小鎮深度遊**：上午科爾馬舊城區深度遊覽（小威尼斯、漁人碼頭、普菲斯泰屋），下午前往埃吉桑姆（法國最美村莊），品嚐亞爾薩斯白酒與火焰薄餅。<br>**住宿**: Hôtel Le Colombier 或 Grand Hotel Bristol |
| **2/6 (五)** | 科爾馬 → 蒂蒂湖 → 加米許 | **黑森林與阿爾卑斯移動日**：⚠️ **08:00 提早出發**，上午遊覽蒂蒂湖（約1.5小時），品嚐黑森林蛋糕、購買咕咕鐘，下午抵達加米許。<br>**住宿**: Das Graseck（連住2晚） |

## 🌍 跨境注意事項（德國 → 法國）

### ✅ 申根區自由通行
- 德國和法國均為申根區成員國，**無邊境檢查**
- 持有效申根簽證或免簽國護照可自由通行

### 🚗 租車保險確認
| 項目 | 確認事項 |
|:---|:---|
| **跨境涵蓋** | ⚠️ 確認 Europcar 保險涵蓋法國境內駕駛 |
| **綠卡 (Green Card)** | 部分租車公司需要，出發前確認 |
| **道路救援** | 確認法國境內救援服務 |
